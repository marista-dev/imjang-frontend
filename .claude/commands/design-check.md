# /design-check — 디자인 품질 점검

모든 페이지를 대상으로 디자인 품질을 점검해.

## 체크리스트

### 모바일 최적화
- [ ] 모든 페이지가 375px에서 깨지지 않는지 확인
- [ ] 터치 타겟이 44px × 44px 이상인지 확인
- [ ] input font-size가 16px 이상인지 (iOS 줌 방지)
- [ ] 하단 콘텐츠가 BottomNav에 가려지지 않는지 (pb-24)
- [ ] safe-area-inset-bottom이 적용되는지

### 디자인 시스템 준수
- [ ] 하드코딩 색상값 없이 Tailwind 토큰만 사용하는지
- [ ] Primary 컬러가 에메랄드 그린(#059669)인지
- [ ] 카드 스타일이 통일되는지 (rounded-2xl, shadow-sm, border)
- [ ] 버튼 스타일이 통일되는지 (h-12, rounded-xl)
- [ ] 폰트가 Pretendard인지

### UI 패턴
- [ ] 빈 상태(Empty State) UI가 있는지
- [ ] 로딩 상태가 있는지 (Spinner)
- [ ] 에러 상태 처리가 있는지
- [ ] 토스트 알림이 적절히 사용되는지

### 시각적 품질
- [ ] 여백이 일관되는지 (px-5, space-y-6)
- [ ] 텍스트 계층이 명확한지 (제목/본문/캡션)
- [ ] 아이콘 크기/스타일이 통일되는지
- [ ] 애니메이션이 자연스러운지

문제 발견 시 즉시 수정하고 수정 내역을 보고해.
